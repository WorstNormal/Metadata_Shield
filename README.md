# Metadata Shield (Детектор ИИ и редактирования)

Metadata Shield — это расширение для Chrome, разработанное для помощи пользователям в идентификации изображений, созданных ИИ или отредактированных в графических редакторах. Оно анализирует метаданные изображений (EXIF, XMP, C2PA) прямо в браузере и предлагает глубокий анализ контента с использованием Google Gemini AI.

## Возможности

*   **Локальный анализ метаданных:** Мгновенно проверяет изображения на наличие следов использования инструментов редактирования (Photoshop, Lightroom) или подписей генерации ИИ (Midjourney, DALL-E, Stable Diffusion), используя данные EXIF/XMP/C2PA.
*   **Глубокий анализ ИИ:** Опциональная интеграция с Google Gemini API для визуального анализа изображений на наличие артефактов ИИ, когда метаданные удалены или не дают однозначного ответа.
*   **Конфиденциальность:** Анализ метаданных происходит полностью локально в вашем браузере. Внешние вызовы API (Gemini) выполняются только тогда, когда вы явно их запускаете.
*   **Гибкие режимы:**
    *   **Авто:** Автоматически сканирует изображения по мере их загрузки.
    *   **Наведение:** Сканирует изображения только при наведении на них курсора.
*   **Визуальные индикаторы:** Четко маркирует изображения с обнаруженными метаданными или подписями ИИ.

## Установка (Режим разработчика)

Поскольку этот проект в настоящее время находится в разработке, вам необходимо собрать его из исходного кода и загрузить как «Распакованное расширение» в Chrome.

### Требования
*   Node.js (v14 или выше)
*   npm

### Шаги сборки

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <repository-url>
    cd Metadata_Shield
    ```

2.  **Установите зависимости:**
    ```bash
    npm install
    ```

3.  **Соберите расширение:**
    ```bash
    # Для продакшн сборки
    npm run build

    # Для разработки (отслеживание изменений)
    npm run watch
    ```
    Это создаст папку `dist`, содержащую скомпилированные файлы расширения.

4.  **Загрузите в Chrome:**
    1.  Откройте Chrome и перейдите по адресу `chrome://extensions/`.
    2.  Включите **Режим разработчика** (переключатель в правом верхнем углу).
    3.  Нажмите **Загрузить распакованное расширение**.
    4.  Выберите папку `dist`, созданную на предыдущем шаге.

## Использование

1.  **Базовое сканирование:**
    *   После установки просматривайте веб-страницы как обычно.
    *   Расширение будет анализировать изображения в зависимости от выбранного режима (Авто или Наведение).
    *   На изображениях появятся значки, указывающие, являются ли они «Созданными ИИ», «Отредактированными» или имеют специфические подписи метаданных.

2.  **Глубокий анализ (Gemini):**
    *   Для использования продвинутого визуального анализа ИИ вам понадобится API-ключ Google Gemini.
    *   Нажмите на значок расширения на панели инструментов, чтобы открыть всплывающее окно.
    *   Введите ваш API-ключ Gemini в настройках.
    *   Наведите курсор на изображение и нажмите значок микроскопа (🔬), чтобы отправить изображение в Gemini для анализа.

3.  **Настройка:**
    *   Откройте всплывающее окно расширения, чтобы:
        *   Включить/отключить расширение глобально.
        *   Переключаться между режимами сканирования «Авто» и «Наведение».
        *   Управлять своим API-ключом.

## Используемые технологии

*   **Manifest V3:** Последняя спецификация расширений Chrome.
*   **exifr:** Для быстрого парсинга метаданных EXIF/XMP/ICC.
*   **c2pa:** Для проверки Content Credentials и данных о происхождении.
*   **Webpack:** Для сборки ресурсов и модулей.
